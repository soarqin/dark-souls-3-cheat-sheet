Cemetery of Ash

  • 出发在第一个转角向右，拾取[弃尸的灵魂]
  • 从喷泉上的尸体拾取[原素灰瓶]
  • 在喷泉后右转，沿着小路穿过一条被水淹没的通道。杀死[贪婪水晶蜥蜴]获取[龙鳞楔形石]然后从附近的尸体上拾取[来历不明旅人的灵魂]
  • 的[斑方石戒指＋1]也在这个区域
  • 回到喷泉，沿着前面的小路走。爬上悬崖，在[灰烬墓地]营火休息，获取[休息]动作
  • 从营火上往下走，从悬崖上跳过去，到棺材上的尸体上拾取一个[楔形石碎片]
  • 从右边的大悬崖上向下走，从附近的尸体上拾取[火焰壶]x5
  • 回到主路，穿过拱门，抽出嵌入石像的剑。杀死[“灰烬审判者”古达]获取[螺旋剑]
  • 恭喜你打败了《黑暗之魂3》的第一个Boss，然后在[“灰烬审判者”古达]营火休息一下
  • 继续穿过大门，从靠在墓碑上的尸体拾取[已折断的直剑]
  • [狼戒指＋2]在你刚经过的门后
  • 前往巨大的悬崖边缘，在山脊的边缘拾取[返回骨片]
  • 沿着左边的小路上悬崖，然后在[传火祭祀场]前面左转从树上的尸体上拾取[双鸟纹木盾]
  • 杀死[专家]得到[打刀]+[专家套装]并激活他的召唤标志
  • 往回走，沿着小路往左走，从[传火祭祀场]入口上方的一具尸体上拾取一个[余火]
  • 继续到另一边拾取一个[返回骨片]从另一个坟墓后面较低山脊的尸体上拾取[余火]
  • 回到主路，然后进入[传火祭祀场]

Firelink Shrine

  • 与[防火女]对话
  • 与[祭祀场侍女]对话
  • 与[脱逃者霍克伍德]对话获取[失意]动作
  • 与[铁匠安德烈]对话获取[欢呼]动作
  • [传火祭祀场]顶部走出去在破旧的小塔旁尸体上拾取[弃尸的灵魂]
  • 如果购买了[塔的钥匙]，打开破旧小塔的门，走上通往钟楼的桥
  • 从桥上向下滚到[传火祭祀场]屋顶，踢下捷径梯子
  • 在屋顶后面杀死[水晶蜥蜴]获取[光辉楔形石]
  • 从屋顶的下半部分拾取[返回骨片]x3
  • 在屋顶里面从横梁上拾取[原素碎片]
  • 与[鸟巢女孩]交易[返回骨片]获取[呼叫]动作和[铁臂套]。关于其他交易选项，请参阅Misc页面
  • 击打横梁后面的[幻影墙]，跳下来从箱子里拾取[贪婪银蛇戒指]
  • 进入钟楼，上电梯，然后[不屈不挠的帕奇]会根据你在游戏中的进展锁上门
  • 在钟楼的顶端，拾取[被污染的防火女灵魂]
  • 回到电梯里，往右边看。小心地跳下墓碑，从尸体上拾取[防火女套装]
  • 到底部，从另一具尸体上拾取[原素戒指]并把门打开

High Wall of Lothric

  • 打开面前的门，在[洛斯里克的高墙]营火休息
  • 沿着右边的小路走到城墙上，然后沿着左边的小楼梯走下去。拾取[弃尸的灵魂]从尸体上
  • 回到楼梯上，在城墙的尽头上下一组楼梯。如果你想[楔形石碎片]+[余火]，等待其中一个不死族转化成[人之脓]然后杀了它。还有，拾取[长弓]+[普通箭]x12从尸体上
  • 回到营火那里，这次沿着左边的路走。拾取[弃尸的灵魂]从尸体上
  • 在城墙的尽头爬上楼梯，再爬上一些楼梯到炮塔的顶部。拾取[望远镜]从尸体上
  • 下楼去，躲在龙后面。进入小门口拾取[黄金松脂]x2从尸体上
  • 进入室内，拾取来[火焰壶]x2从桌子旁的尸体上
  • 走下梯子，拾取[弃尸的灵魂]从牢房外的尸体上找到的
  • 狙击龙，直到它掉落[楔形石大碎片]飞走了
  • 拾取[大剑]，[棍棒]，[余火]和[弃尸的大块灵魂]从龙的前面
  • 打开龙所在位置下面的门。下楼。要么杀了[贪欲者/宝箱怪]或者用一个[猎捕不死人护符]为了得到[幽邃战斧]
  • 从另一边的门出去，拾取[弃尸的灵魂]从左边的尸体上
  • 到下一栋楼去接电话[火焰壶]x3从木梁上
  • 从侧门出去，上楼梯，拾取[楔形石碎片]从尸体上。点燃[高墙边塔]营火
  • 点燃[高墙边塔]营火后，返回[传火祭祀场]和他交谈[无名指的李奥纳德]谁给你的?[龟裂血红眼眸宝珠]x5． 他会在洛思里克的王座旁。如果他现在还没有，那就杀了沃特再检查一下
  • 回到大楼里走下楼梯，从侧门出去，拾取一个[弃尸的灵魂]从右边的尸体上
  • 到屋顶上去。如果你想[楔形石碎片]+[余火]，等待其中一个不死族转化成[人之脓]然后杀了它。然后拾取来[火焰壶]x3从尸体上。继续前进，杀死[水晶蜥蜴]接受[粗制宝石]． 屋顶上还有一具尸体[弃尸的大块灵魂]
  • 的[咬人戒指＋1]是不是在另一个可以跳上去的屋顶上
  • 把梯子拿下来，到侧屋顶上去拿[黑火焰壶]x3从尸体上
  • 在相邻的阳台上与弓箭手一起拾取[火焰壶]x3从尸体上
  • 回到之前的那栋楼，梯子旁边。往左走，滚进锅里去[猎捕不死人护符]x2从尸体上。继续进去，拾取[楔形石碎片]在小心伏击的同时从尸体上找到
  • 回到主房间，拾取[弃尸的灵魂]藏在第一个角落里的尸体
  • 继续穿过门口，走左边的下一个门。拾取[阔剑]从尸体上
  • 沿着正确的路走下楼梯，找到[银鹫鸢形盾]从胸部开始
  • 回到原来的入口，发现门口被碎片堵住了。滚过去拾取来[绿花草]x2在窗台的尽头
  • 下拉，拾取[亚斯特拉直剑]从胸部开始
  • 继续进入房间，拾取[监牢的钥匙]和[楔形石碎片]房间里的尸体
  • 拾取一个[原素碎片]在同一个房间的祭坛上
  • 的[邪眼戒指＋2]是在房间后面的桶后面吗
  • 从较低的一层出去，然后往下走，拾取[细剑]在排水沟旁边
  • 你可以在喷泉周围的尸体上收集[余火]和另一个[余火]
  • 往院子里走，左转拾取[弃尸的灵魂]从一具尸体上
  • 在庭院入口的右边上楼，沿着小路直走。离开后第一个门立即左转去找[绿花草]x3在尸体上
  • 启动电梯，从长城营火处打开一条捷径。电梯的后面是[飞刀]x6在尸体上
  • 旅行到[高墙边塔]，并继续在塔内的所有方式。到隔壁房间去拿[飞刀]x8在尸体上
  • 沿着通道继续走，然后拾取[破甲细剑]门对面是一具尸体
  • 再往下走，你会发现一间锁着的牢房。与[监牢的钥匙]你可以自由[不死聚落的葛雷瑞特]和他谈谈。如果你同意找到洛丽塔，他就给你[蓝泪石戒指]． 然后他转到[传火祭祀场]
  • 旅行到[洛斯里克的高墙]，乘电梯走捷径下来。回到主庭院，但这次走左边的楼梯，俯瞰喷泉区。拾取[弃尸的大块灵魂]在屋顶上
  • 跳上遮阳篷捡一个[牺牲戒指]
  • 继续回到庭院。在露台的角落里，就在通往楼下的楼梯前，你可以召唤[狮子骑士艾伯特]
  • 向左转。沿着城堡前左边的楼梯往上走，杀死红眼怪[洛斯里克骑士]为了得到一个[熟练宝石]
  • 在教堂里和[主祭艾玛]获取[洛斯里克小环旗]． 再和她谈谈[青教]约
  • 从教堂出来走左边的楼梯然后拾取[鹤嘴戟]从尸体上
  • 在穿过大门之前，你可以召唤[专家]如果他早些时候在[传火祭祀场]被杀
  • 穿过大门，击杀敌人[冷冽谷的玻尔多];收到[冷冽谷的玻尔多的灵魂]． 杀死Vordt后，在悬崖上举起旗帜进入[不死聚落]

Undead Settlement

  • 在蝙蝠翼魔把你放下后，走到左边，拾取一个[弃尸的大块灵魂]
  • 下楼。走在一辆翻倒的马车后面，杀死两个人[狗]得到[诱敌头盖骨]x2
  • 找到[隆道尔的尤艾尔]在营火下和其他几个朝圣者一起。与他交谈并同意他的请求，他移动到[传火祭祀场]。返回到[传火祭祀场]并与[隆道尔的尤艾尔]交谈以获取[招手]动作。接受约尔的提议，抽出真正的力量，你将获得一个免费等级和一个[黑暗印记]． 从现在起，你每死亡一次，每[黑暗印记]获得1个空化。[隆道尔的尤艾尔]在2、6、12和15镂空提供了4个免费级别，所以获得所有5个黑暗印记应该不会花很长时间。如果你在到达[卡萨斯的地下墓地]之前没有得到全部5个黑暗印记，约尔死亡，夺火结局失效
  • 当你得到所有5个黑暗印记并刷新世界后，[隆道尔的尤艾尔]死于[隆道尔的尤莉雅]看着他新发现的尸体。与她交谈，接受高贵的[鞠躬]动作并购买[隆道尔点字圣典]
  • 在约尔附近，你会发现[返回骨片]x2在桥的最左边
  • 从[不死聚落]营火，进入房子。砍倒一具发光的挂尸，得到[小皮盾]
  • 下楼去拿[散装焦炭松脂]x2． 走到阳台上，把吊着的尸体砍下来。右转到拐角处去[修理光粉]x2
  • 下楼再来一杯[散装焦炭松脂]x2
  • 到外面去拿[罗蕾塔的骨头]从你砍下的尸体上
  • 拾取一个[原素碎片]在燃烧的树旁，有崇拜亡灵的人
  • 在树的另一边，抓一个[余火]． 把一具挂在树上的尸体射下来[廓尔喀弯刀]x9． 走到关着的门口去拿个[来历不明旅人的灵魂]
  • 走到房子的右边抓一个[楔形石碎片]
  • 穿过房子的一个大拱门，打开一扇门，走进房间。拾取[焦炭松脂]x2从墙边的一具尸体上。把挂在楼梯上的尸体砍下来[弃尸的大块灵魂]． 从房间的洞里掉下来。在这个隐藏的房间里，发现[太阳战士]契约和消费[原素汤]
  • 打开门，走到走廊的尽头。你现在应该在附近的主要街道上了。向右走，穿过你右边第一个缺口之间的木箱。拾取一个[来历不明旅人的灵魂]
  • 沿着主街往回走，继续往前走。木门应该自动打开。走进那所房子去拿[软鞭]
  • 继续走回主街。一直走，直到你看到右手边有一个木轮。走到拐角处找辆[楔形石碎片]
  • 爬上阳台[导师]． 拾取一个[楔形石碎片]
  • 从阳台爬下来，走到屋顶上。跳上你左边的木制平台去抓[锈斑硬币]x2
  • 跳下去杀了[水晶蜥蜴]这就降低了[锋利宝石]． 在下面点燃[颓旧桥端]营火
  • 疯狂的黑暗精灵[圣骑士佛多林克]如果你有余烬，你将会在[颓旧桥端]营火附近入侵杀了他[枷锁脊椎骨]
  • 回到[不死聚落]开始附近的燃烧的树，走过桥，走在右边应该被木箱和木桶阻挡。朝房子走去[奴隶]． 找到一个[即将消逝的灵魂]在这里。绕着房子走一圈，去拿[木板盾]． 走到房子的屋顶到你的左边，拾取[火焰壶]x6
  • 爬到另一个房子的屋顶上抓[返回骨片]x2． 跳下去到一个没有敌意的笼子搬运工正在走动的地方。砍倒最后的吊尸，拾取[火方石戒指]． (如果尸体倒了，你无法得到物品，就进入下一步)
  • 从后面检查运笼机。走出牢笼后，向光明走去。拾取[战神木盾]． 跟[圣骑士佛多林克]获取[累积者]契约与契约[返回骨片]． (如果你到达[咒蚀大树]之前到达笼载者，[圣骑士佛多林克]死亡，笼载者变得敌对。因此，你不能获得契约，直到后来通过[薄暮之国的希里丝]的支线任务。)退出并重新装填以在笼子运输船附近出现。如果你之前没拿到戒指，就把它找回来
  • 往回走，沿着左边的一个小岩架往下走。进入桥附近的房子，拾取一个[弃尸的大块灵魂]． 然后打开后门
  • 从你的右边，得到[孪生蛇纹圆盾]
  • 穿过桥，朝你看到的房子走去，然后跳到你右边的窗台上。当你站在窗台上时，向桥那边走[楔形石碎片]
  • 另一条路会把你带到房子下面的营火上，然后往上走
  • 走到房子的顶部，找到一个笼子，里面装着[大沼的柯弭库斯]． 与他交谈，他移动到[传火祭祀场]。在神殿，和他多次交谈以获得你的[咒术之火](除非你已经是一个纵火术士)和[欢迎]动作
  • 拾取[手斧]然后得到一个[游击枪]从附近一具吊着的尸体上
  • 从房子里走下来，走到绞刑架上。拾取一个[来历不明旅人的灵魂]． 步行穿过未完工的桥去[火焰怪手戒指]
  • 穿过房子的拱门，走下楼梯。往左走，拐个弯就到了[弃尸的大块灵魂]． 走下楼，朝一棵树走去[狗]． 杀了它，拿到[余火]
  • 往桥那边跑，然后进入右边的下水道。杀死一个巨人[老鼠]为了得到[咬血戒指]． 拾取[护手带]爬上去打开捷径
  • 穿过下水道走回去，到桥上再找一个[余火]． 上楼朝你右边的房子走去。拾取一个[弃尸的大块灵魂]． 绕着房子的右边走[诱敌头盖骨]x3
  • 朝塔楼走去。跟[卡利姆的伊果]在外面。如果你有足够的血量，你可以掉到山谷里去[卡利姆的伊莉娜]在她的牢房里，不过不用麻烦了。你可以通过从神社少女那里购买[墓地的钥匙]到达伊琳娜[安葬者骨灰]在这个列表中哪个更靠后
  • 走进高塔去找[卡塔利纳的杰克巴尔多]． 坐电梯去地下室。杀死[冷冽征战骑士]为了得到[伊鲁席尔直剑]，拾取[余火]然后休息[活祭品之路]营火
  • 的[生命戒指＋1]在西格沃德后面的阳台上
  • 要到达塔顶，在主入口按下电梯的开关，然后迅速下车，就会看到另一部电梯。
  • 在电梯顶端下电梯后，你会看到一个[无名士兵的灵魂]在角落里。
  • 上楼，向不死聚落的巨人献上友谊的象征这样他就不会用巨大的箭攻击你，甚至在你靠近白桦树时帮助你。
  • 乘电梯下去，面对着塔的入口，当你听到西格沃德的声音时;翻滚着降落在一个木制平台上。
  • 出去再和齐格沃德谈谈。
  • 引诱火妖向塔，使西格沃德来战斗在你身边。
  • 杀死火魔后，你将获得[火焰宝石].

FAQ

The foundation of this guide was based on Stephen McNabb's Dark Soul 2 Cheat Sheet source code and DeathGodGarra's NPC Side Quests Guide V2 walkthrough guide

This guide has evolved thanks to the many GitHub contributors with many great suggestions, fixes and improvements!

I have feedback, how can I contribute?

You can visit the GitHub repository and report Issues or create a fork and submit a Pull Request.

Is this a guide?

Nope, not really. This is a set of checklists and information that you can use while playing Dark Souls to make sure you don't miss an item, action, conversation, or boss. If you need playthroughs then you can click on the zone titles to read the wiki walkthrough.

Do I need to follow the playthrough in this exact order?

Dark Souls 3 is not a linear game and has multiple ways of progressing. This guide highlights zones as they become available, so feel free to follow however you please. You can always check things off in a different order.

Can I use this for multiple characters?

Yup, use the profile selector and buttons in the options tab at the top of the page to setup multiple profiles.

How does the checklist status get saved?

The checklist is saved to your browser's local storage. Be careful when clearing your browser's cache as it will also destroy your saved progress.
